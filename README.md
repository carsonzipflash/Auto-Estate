# Prophet Homes Dashboard

Internal real estate dashboard for Prophet Homes brokerage. Built with Next.js 14 (App Router), TypeScript, and Tailwind CSS.

## Modules

| Route | Description |
|---|---|
| `/pipeline` | Deal Pipeline — motivated seller lead tracking |
| `/underwriting` | Comp & Underwriting — comparable sales and deal analysis |
| `/map` | Map View — geographic visualization of leads and properties |

## Running locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Linting:** ESLint
